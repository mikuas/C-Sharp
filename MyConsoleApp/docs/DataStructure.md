### 数据结构(System.Collection)


| 数据结构                           | 特点         | 适用场景                 |
|--------------------------------|------------|----------------------|
| **`Array`**                    | 固定大小，快速访问  | 元素数量固定且访问模式简单的场景     |
| **`List<T>`**                  | 动态大小，按索引访问 | 需要动态增删和顺序访问的场景       |
| **`Dictionary<TKey, TValue>`** | 键值对存储，快速查找 | 快速查找数据，键唯一的场景（如配置管理） |
| **`HashSet<T>`**               | 唯一元素，集合操作  | 不需要重复元素，快速查找和去重的场景   |
| **`SortedList<TKey, TValue>`** | 自动排序的字典    | 需要按键排序的场景            |
| **`Queue<T>`**                 | 先进先出（FIFO） | 排队场景（如任务处理，打印队列）     |
| **`Stack<T>`**                 | 后进先出（LIFO） | 栈操作（如递归，函数调用栈）       |
|                                |            |                      |
