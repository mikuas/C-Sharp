## 文件File IO(System.IO)

### `FileMode` 枚举

| 枚举值            | 说明                                  |
|----------------|-------------------------------------|
| `Create`       | 如果文件不存在，则创建新文件；如果文件已存在，则覆盖文件。       |
| `CreateNew`    | 如果文件不存在，则创建新文件；如果文件已存在，则引发异常。       |
| `Open`         | 只打开已存在的文件。如果文件不存在，则引发异常。            |
| `OpenOrCreate` | 如果文件不存在，则创建新文件；如果文件已存在，则打开文件。       |
| `Append`       | 如果文件不存在，则创建新文件；如果文件已存在，则将数据追加到文件末尾。 |
| `Truncate`     | 如果文件存在，则打开并清空文件内容。如果文件不存在，则引发异常。    |

### `FileAccess` 枚举

| 枚举值         | 说明                          |
|-------------|-----------------------------|
| `Read`      | 只读访问权限。只能读取文件内容，不能写入文件。     |
| `Write`     | 只写访问权限。只能写入文件内容，不能读取文件。     |
| `ReadWrite` | 读写访问权限。既可以读取文件内容，也可以写入文件内容。 |
